MarkDown基本语法

一、标题

# 一级目录
## 二级目录
### 三级目录
#### 四级目录

二、段落

换行：

markdown  <-结尾跟两个空格
markdown2

或者

markdown
            <-使用空行来换行
markdown2

字体：

*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___

分隔线：
使用三个或三个以上的星号、减号、底线，行内不能有其他的东西。

***
* * *
*******
---
______

删除线:

~~内容~~

内容

下划线:

带下划线的文本

带下划线的文本

三、列表

无序列表 

*
-
+

有序列表 

1. 第一项
2. 第二项
3. 第三项

嵌套列表
在子列表前添加四个空格 

1. 第一项
    - 子项
2. 第二项
    - 子项
  1. 第一项
    • 嵌套子项
  2. 第二项
    • 嵌套子项

四、区块

引用

> 一级引用内容
>> 二级引用内容,以此类推

引用一

引用二

区块中使用列表

> 区块中使用列表
> 1. 有序
> + 无序

区块中使用列表

  1. 有序
  • 无序

列表中使用区块
需要在>前添加四个空格的缩进

* 第一项
    > 引用一
    >> 引用二
* 第二项
  • 第一项

    引用一

    引用二

  • 第二项

五、代码

`代码片段`(反引号)

\```java(可以指定一种语言,也可以不指定)
代码区(三个反引号)
\```

代码片段:def hello(),
代码区:

def hello():
    print("hello")

代码区也可以使用四个空格或一个制表符

六、连接

普通链接

[连接名称](链接地址)
或
<连接地址>

百度 或 http://www.baidu.com/

图片

![alt 属性文本](图片地址)
![alt 属性文本](图片地址 "可选标题")

七、表格

使用|来分割不同的单元格,使用-来分割表头和其他行

|  表头  |  表格  |  表格  |
| :----- | :-----:| -----: |
| 左对齐 | 居中   | 右对齐 |
| 单元格 | 单元格 | 单元格 |
表头 表格 表格
左对齐 居中 右对齐
单元格 单元格 单元格

八、字体

我是黑体字
我是微软雅黑
我是华文彩云
我是红色
我是绿色
我是蓝色
我是尺寸
我是黑体，绿色，尺寸为5
